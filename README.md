# Recipe_Sharing_System_Using_Angular
🍳✨ Recipe Sharing System Using Angular – A sleek &amp; interactive platform where chefs 👨‍🍳 can create, edit, and showcase recipes. Features include secure login 🔐, ratings ⭐, comments 💬, admin control ⚙️, and smart dashboards 📊 for a delightful cooking community! 🚀🔥

## 📂 Project Highlights

✔️ Standalone Angular Components  
✔️ Signal-based state management  
✔️ Modular & clean architecture  
✔️ Real-time UI updates  

## 🚀 Core Functionalities

### 🔐 Authentication System
- Admin login with predefined credentials
- Chef registration & login
- Session stored using LocalStorage 💾
- Role-based access (Admin / Chef)

---

### 👨‍🍳 Chef Features

🍽️ **Recipe Management**
- Add new recipes (title, description, ingredients, steps, image)
- Edit existing recipes
- Delete own recipes

🔍 **Search Functionality**
- Search recipes by title in real-time

⭐ **Ratings System**
- Rate recipes (1–5)
- One rating per chef per recipe
- View average ratings

💬 **Comments System**
- Add comments to recipes
- View other chefs’ feedback

📂 **Recipe Filtering**
- View "My Recipes"
- View "Other Chefs' Recipes"

---

### 📊 Admin Features

⚙️ **Chef Management**
- View all registered chefs
- Delete chefs (removes all their recipes)

🗑️ **Recipe Control**
- Delete any recipe from the system
